E-Wallet Transaction System

A robust digital wallet platform designed to handle secure and seamless monetary transactions. The system provides RESTful APIs for wallet recharge, fund transfer, and transaction history tracking. Built with a strong focus on data consistency and reliability, the application ensures ACID-compliant wallet operations using Hibernate and transactional management.

Key Features

Wallet Recharge: Allows users to add funds to their digital wallet securely.
Fund Transfer: Supports wallet-to-wallet money transfers with balance validation and transaction logging.
Transaction Tracking: Provides detailed transaction history with timestamps, types, and amounts.
ACID Compliance: Guaranteed consistency and atomicity of financial operations through Hibernate and Spring transaction management.
Secure APIs: Input validation and error handling for failed transactions, insufficient balance, and invalid wallet IDs.
Technologies Used

Backend: Spring Boot ,
Database: MySQL with Hibernate ORM ,
Architecture: RESTful API design ,
Tools: Maven, Postman .
